# 🖋️✨ RyanScript_Fonts ✨🖋️

## 📌 Overview

This repository contains all files related to the **Ryan Script Regular** font created by **Ryan Pereira**. The repository is organized into folders and files that contain the font assets, documentation, and supporting materials.

This project is designed to showcase custom typography design, handwritten font styling, and creative font development.

## ▶️ How to Use

Clone the repository:

```bash
git clone <your-repository-url>
```

Navigate into the folder:

```bash
cd RyanScript_Fonts
```

Open the font folder:

```bash
cd RyanScript-Regular
```

Install the font:

### 🖥️ Windows

Right-click:

```text
Ryanscript-Regular.ttf
```

Then click:

```text
Install
```

### 🍎 macOS

Open the font file and press:

```text
Install Font
```

### 🐧 Linux

```bash
mkdir -p ~/.fonts
cp Ryanscript-Regular.ttf ~/.fonts
fc-cache -fv
```
